If your API key might be compromised, change your API key:

1.  Log in to the [Cloudflare dashboard](https://dash.cloudflare.com).
2.  Go to **Profile** > **API Tokens**.
4.  In the **API Keys** section, find your key.
5.  Select **Change**.
